# Unity MCP Token Bridge

Client package for Unity MCP Monetization.
Connects your Unity Editor to the Monetization Server and provides an IDE Bridge for AI Agents.

## Setup
1. Import this package.
2. Go to `Tools > MCP Token Manager`.
3. Enter your API Key (get it from the Telegram Bot).
4. Click "Generate IDE Config" and paste it into your Windsurf/Cursor config.
